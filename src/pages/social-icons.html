Some pretty social icons to enhance your project. Adam Fairhead created a set of sweet resolution-independent SVG icons. These SVG icons are great for ultra-high resolution screens, lighter weight, and customizeable.

Why Do Resolutions Matter?

Just like Foundation, these icons are pushing the web in new directions. These icons are a great resource for iconography that cannot be affected by how often pixel densities change. The use of SVG will continue to grow across the web, be part of the movement.

Download the code from the Playground
Not Tested in 5

We made these add-ons to be pretty robust, but they have not been tested in Foundation 5 yet. Watch out for gremlins if you're using Icon Fonts, Responsive Tables or the SVG Social Icons with Foundation 5.
