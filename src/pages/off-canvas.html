Off Canvas Layouts

Get the flexibility to hide things off canvas and find them when you need them.

What's an Off Canvas Layout?

You know that nifty little menu hidden under the main Facebook app feed? Or the one that lives underneath the Path feed that reveals your friends list. Luke Wroblewski and Jason Weaver came up with these bad boys and we adapted them for Foundation so that you can use them today!

Download the code from the Playground
Not Tested in 4

We made these add-ons to be pretty robust, but they have not been tested in Foundation 4 yet. Watch out for gremlins if you're using Icon Fonts, Off-Canvas Layouts, Responsive Tables or the SVG Social Icons with Foundation 4.
